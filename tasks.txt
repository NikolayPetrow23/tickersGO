Функции приложения

  - Выбор/сохранение билета (файла/пути к файлу) Будет копирование + сохранение пути к файлу
  - Парсинг файла (комания, дата, qr/штрих код) Пробовать через нативные инструменты
  - Сохранение данных из парсинга для быстрого доступа (штрих код, кур код) ОК 
  - Ввод инфы (подумать над полями) 
  - Напоминания/уведомления
  - Вход в аккаунт (хранение токена), вход через акк apple, google и telegram
  - Синк между бд, как вариант тупо хранить бд
 
RN
  - UI
  - Нативные модули: Сканер камеры, OCR (Сканер текста), PDFKit, сканер barcode, Выбор файла и тд
  - Вызов GO Core

GO Core
  - Бизнес логика в приложении (модельки, запросы, auth, парсинг/валидация)
  - Многопоточка

GO Backend
  - Бизнес логика всего проекта (Пользователи, синк, разные модельки и тд!)
  - Auth и тд
  - Прокси с LLM (Запрос с промптом, валидация и передача обратно)
